# mochi_studio

Interactive GraphQL studio for mochi — playground, query collections, and visual schema explorer.

## Installation

```toml
# gleam.toml
[dependencies]
mochi_studio = { git = "https://github.com/qwexvf/mochi_studio", ref = "main" }
```
